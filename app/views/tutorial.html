Topic models are not always perfect. Sometimes we find issues like words or documents that don’t make sense with the rest of the topic or see places where some words might be missing. We might even find that a topic the system has identified that seems like it’s talking about two distinct things, so should be split, or that multiple topics seem to be talking about the same thing, so they should be merged. To fix these and other problems, the interface lets you make changes to the model.
